#JSF, JPA com Hibernate e Spring (Fase 5)
#Descrição
Estrutura para projetos JSF, incluindo crud de usuário save() implementado.

#Requisitos
- Java 7
- Tomcat 8
- Maven 4

#Frameworks Utilizados
- JSF 2.2
- Spring 4.1
- Servlet 3.1
- Primefaces 5.2
- Hibernate 5.1
- Jpa 2.1

#Import
Importar como maven project.
